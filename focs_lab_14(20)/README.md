## 7. Сценарий выполнения работы

**Команды для обработки файлов**

| _Команда_ | _Флаг_ |_Назначение команды_ |
| --- | --- | --- |
| **``diff``** | ---  | Выводит различие между двумя файлами|
| --- | ``-u`` or ``--unified`` | Использует формат вывода в стиле "Unified diff" |
| --- | ``-c`` or ``--context`` | Использует формат вывода в стиле "Context diff" |
| --- | ``-r`` or ``--recursive`` | Рекурсивно сравнивает директории |
| --- | ``-q`` or ``--brief`` | Выводит только информацию о различиях, без подробных изменений |
| --- | ``-i`` or ``--ignore-case`` | Игнорирует регистр букв при сравнении |
| --- | ``-w`` or ``--ignore-all-space`` | Игнорирует все пробелы при сравнении |
| --- | ``-B`` or ``--ignore-blank-lines`` | Игнорирует все пустые строки при сравнении |
| --- | ``-S`` or ``--ignore-file-name-case`` | Игнорирует регистр имен файлов при сравнении |
| --- | ``-N`` or ``--new-file`` | Выводит информацию о новых файлах |
| --- | ``-p`` or ``--show-c-function`` | Выводит информацию о функциях C, если файлы были изменены внутри них |
| --- | ``-y`` or ``--side-by-side`` | Использует формат вывода "Side-by-side diff" |
| --- | ``-l`` or ``--paginate`` | Пагинирует вывод результатов |
| --- | ``--suppress-common-lines`` | Не выводит строки, которые не отличаются в сравниваемых файлах 
| **``file``**| --- | Определяет тип файла |
| --- | ``-b`` | выводит только имя файла без дополнительной информации |
| --- | ``-c`` | выводит содержимое файла, если это обычный текстовый файл |
| --- | ``-z`` | выводит нулевой символ вместо перевода строки в конце вывода |
| --- | ``-i`` | игнорировать маленькие ошибки в формате файла |
| --- | ``-m`` | использовать магические данные для определения типа файла вместо только расширения |
| --- | ``-k`` | не использовать системную базу данных магических чисел |
| --- | ``-L`` | следовать символическим ссылкам |
| --- | ``-P`` | не следовать символическим ссылкам |
| --- | ``-s`` | выводит только имя файла и описание типа файла в одной строке |
| --- | ``-v`` | выводит подробную информацию о версии программы |
| --- | ``-h`` | выводит краткую справку о программе |
| **``patch``** | --- | Применяет разницу между двумя файлами к первому файлу |
| --- | ``-pNUM`` | Определяет количество уровней в исходном пути, которые будут игнорироваться |
| --- | ``-i FILE`` | применить изменения из указанного файла вместо стандартного ввода |
| --- | ``-o FILE`` | записать результат в указанный файл вместо стандартного вывода |
| --- | ``-R`` | восстановить файл, используя измененный файл и оригинал |
| --- | ``-s`` | тихий режим. Не выводить сообщения о применении патча |
| --- | ``-u`` | формат вывода в стиле diff вместо стандартного формата |
| --- | ``-z`` | разделение патча на записи с нулевым байтом вместо символа перевода строки |
| **``dd``** |  ---  | Конвертация и копирование файла  |
| **``gzip``** |  ---  | Сжатие и распаковка файлов |
| **``bzip2``** |  ---  | Сжатие файлов |
| **``wc``** | ---  |Подсчет количество строк, слов, символов и байтов каждого заданного файла или ввода |
| **``comm``** |  ---  |Сравнение двух отсортированных файлов построчно|
| **``grep``** |  ---  |Поиск строк, соответствующих строке в тексте или содержимому файла|
| **``sort``** |  ---  |Сортировка, объединение или проверка последовательности текстовых файлов |
| **``tail``** | ---  | Вывод последней части файлов |
| **``tar``** | ---  | Архивирование файлов |
| **``cmp``** |  ---  |Побайтовое сравнение двух текстов файлов |
| **``cut``** | ---  | Позволяет вырезать часть строк из указанных файлов или данных по конвейеру |
| **``od``** | ---  | Преобразование содержимого входных данных в различные форматы с восьмеричным форматом по умолчанию|
| **``iconv``** | ---  | Преобразование текста из одной кодировки символов в другую |
| **``df``** | ---  | Отображение информации, связанной с файловыми системами, об общем объеме и доступном пространстве |
| **``cpp``** | ---  | Препроцессор языка C |
| **``xargs``** | ---  | Сборка и выполнение командных строк из стандартного ввода |
| **``paste``** | ---  | Объединение строк файлов по горизонтали|
| **``ed``** | ---  | Запуск текстового редактора ed |
| **``tee``** | ---  | Считывание стандартного ввода и записывание его как в стандартный вывод |
| **``head``** | ---  | Вывод верхнего числа N данных заданного ввода |
| **``split``** | ---  | Разделение больших файлов на файлы меньшего размера |
| **``sum``** | ---  | Поиск контрольной суммы и подсчет блоков в файле|
